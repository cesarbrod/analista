# O Analista de Bagé!

## Converse com um psicólogo muito freudiano adepto à cura pelo joelhaço.

Esse programa foi criado como um exercício para a Imersão em IA promovida pela Alura em parceria com o Google, em especial com o que foi ensinado na Aula 4.

O programa analista.py, nessa mesma pasta, está devidamente comentado para que você possa exercitar suas modificações. Brinque a vontade!

Para rodar a versão no terminal, certifique-se antes de ter instalado o python e os seguintes pacotes:

google-genai

Instale com 

`pip install google-genai`

Execute o comando:

`python3 analista.py`

Para ver o programa rodando na web acesse http://brodtec.com/analista
